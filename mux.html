FAQ

What is a multiplexer?

A multiplexer (MUX) selects one of several input signals and forwards it to a single output line. The selection is controlled by select lines. A 4:1 MUX has 4 inputs, 2 select lines, and 1 output.

What is a demultiplexer?

A demultiplexer (DEMUX) does the opposite — it takes one input and routes it to one of several outputs based on select lines. A 1:4 DEMUX has 1 input, 2 select lines, and 4 outputs.

How are MUX/DEMUX used in practice?

Multiplexers are used in data routing, communication systems, memory addressing, and implementing Boolean functions. They are fundamental building blocks in digital systems.
