FAQ

Why convert to NAND-only?

NAND is a universal gate — any Boolean function can be built using only NAND gates. This simplifies manufacturing since only one type of gate is needed on the chip.

How does the conversion work?

Using De Morgan's theorems: NOT A = A NAND A, A AND B = (A NAND B) NAND (A NAND B), A OR B = (A NAND A) NAND (B NAND B).

Is NAND or NOR more common in real circuits?

Both are universal gates. NAND is more common in CMOS technology because NAND gates are faster and require fewer transistors than NOR gates in most implementations.
