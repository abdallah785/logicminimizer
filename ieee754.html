FAQ

What is IEEE 754?

IEEE 754 is the standard for floating-point arithmetic used by virtually all modern CPUs. It defines how decimal numbers are stored in binary using sign, exponent, and mantissa fields.

What is the difference between single and double precision?

Single precision uses 32 bits (1 sign + 8 exponent + 23 mantissa) with ~7 decimal digits of precision. Double precision uses 64 bits (1 sign + 11 exponent + 52 mantissa) with ~15 decimal digits.

Why does 0.1 have no exact IEEE 754 representation?

0.1 in decimal is a repeating fraction in binary (0.0001100110011...), similar to how 1/3 repeats in decimal. This causes floating-point rounding errors in computations.

What are special IEEE 754 values?

IEEE 754 defines special values: +0 and -0, positive and negative infinity (exponent all 1s, mantissa all 0s), and NaN (exponent all 1s, mantissa non-zero).
