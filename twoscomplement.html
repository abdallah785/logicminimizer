FAQ

What is two's complement?

Two's complement is the standard method for representing signed integers in binary. The most significant bit indicates the sign: 0 for positive, 1 for negative.

How do you convert a negative number to two's complement?

First write the absolute value in binary, then invert all bits (one's complement), and finally add 1. For example, -42 in 8-bit: 42 = 00101010, invert = 11010101, add 1 = 11010110.

What is the range of an 8-bit two's complement number?

An 8-bit two's complement can represent values from -128 to +127. For n bits, the range is -(2^(n-1)) to +(2^(n-1) - 1).

Why is two's complement used instead of sign-magnitude?

Two's complement allows addition and subtraction to use the same circuit, has only one representation of zero, and simplifies hardware design in CPUs and ALUs.
