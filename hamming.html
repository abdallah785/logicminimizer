FAQ

What is Hamming code?

Hamming code is an error-correcting code that can detect and correct single-bit errors. It works by adding parity bits at positions that are powers of 2 (1, 2, 4, 8, etc.).

How many parity bits are needed?

For m data bits, you need r parity bits where 2^r >= m + r + 1. For 4 data bits, you need 3 parity bits, giving a 7-bit Hamming code.

Can Hamming code correct multiple errors?

Standard Hamming code can only detect and correct single-bit errors. SEC-DED (Single Error Correction, Double Error Detection) adds an extra parity bit to detect double errors.
